    API Access

    Como obter API ID e API Hash

    1. Acesse my.telegram.org/apps.
    2. Faça login com o mesmo número de telefone da sua conta Telegram.
    3. Abra a seção API development tools.
    4. Crie um app preenchendo nome, short name e plataforma.
    5. Copie o api_id e o api_hash exibidos.
    6. Volte ao Telegram Drive e preencha os campos de login.
